Sugestão de projetos para ser desenvolvidos:


1. Sistema de Gestão Acadêmica
Objetivo: Simular um sistema de universidade (alunos, disciplinas, notas, inscrições).

Tecnologias:

Back-end: Node.js/Express ou Django.

Banco de dados: PostgreSQL/MySQL.

Front-end: React ou Vue.

Conceitos aplicados:

Transações e concorrência (ex: inscrição simultânea em disciplinas).

Níveis de isolamento (dirty read, phantom read).

Interface moderna com dashboards.




2. Plataforma de E-commerce Acadêmico
Objetivo: Loja online para venda de livros ou materiais de estudo.

Tecnologias:

Front-end: React + TailwindCSS.

Back-end: Node.js + API REST.

Banco de dados: MongoDB ou SQL.

Conceitos aplicados:

CRUD completo (produtos, carrinho, pedidos).

Autenticação e autorização.

Copy e design alinhados ao tema educacional.





3. Aplicação de Simulação de Concorrência
Objetivo: Criar um site que demonstra anomalias de transações (deadlock, lost-update, phantom read).

Tecnologias:

Back-end: Java (Spring Boot) ou Node.js..

Banco de dados: PostgreSQL.

Front-end: Interface interativa para visualizar cenários.

Conceitos aplicados:

Engenharia de software + bases de dados.

Demonstração prática de teoria aprendida na licenciatura.




4. Portal de Projetos Acadêmicos
Objetivo: Site para publicar e organizar projetos da licenciatura.

Tecnologias:

Front-end: Next.js..

Back-end: API em Node.js/Django..

Banco de dados: SQLite/PostgreSQL.

Funcionalidades:

Upload de relatórios e código.

Tags e categorias por disciplina.

Área de contato e networking.
